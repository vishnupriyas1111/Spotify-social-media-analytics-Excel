# Spotify Social Media Analytics - Excel Project

## Project Overview
This project analyzes Spotify’s social media performance across multiple platforms to generate insights for marketing strategy.

## Objective
- Identify high-performing content and platforms
- Analyze engagement trends
- Evaluate campaign effectiveness
- Optimize platform and ad spend strategy

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts & Dashboards

##  Project Structure
- Task 1: Data Preprocessing & Cleaning
- Task 2: Engagement Analysis
- Task 3: Platform Performance Analysis
- Task 4: Hashtag & Content Strategy
- Task 5: Campaign Effectiveness
- Task 6: Follower Analysis
- Task 7: Video Explanation

## Key Insights
- Twitter achieved the highest engagemnt rate (9.60 %) making it the most effective platform for user interaction.
- #SpotifyWrapped is the top-performing hashtag based average clicks, indicating strong audience interest.
- Instagram shows the most consistent weekly follwer growth rate,making it a reliable platform for audeince expansion.
- Analysis suggests focusing primarily on Twitter and Instagram, while maintaining limited presence on Youtube for brand visibility.
- Certain content type-platform combinations delivered significantly higher engagement.
- Identified the campaign with the highest ROI of 0.0465, providing insights into cost-effective marketing strategies.
- Moving average indicates relatively stable follower growth over the period,suggesting strong audience retention.
- No strong correlation found between ad spend and engagement, implying that engagement is influenced by other factors as well.
