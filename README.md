# Spotify Social Media Analytics - Excel Project

## Project Overview
This project analyzes Spotify’s social media performance across multiple platforms to generate insights for marketing strategy.

## Objective
- Identify high-performing content and platforms
- Analyze engagement trends
- Evaluate campaign effectiveness
- Optimize platform and ad spend strategy

## Tools & Techniques
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Analysis
- Dashboard Creation

## Project Structure
- Task 1: Data Preprocessing & Cleaning
- Task 2: Engagement Analysis
- Task 3: Platform Performance Analysis
- Task 4: Hashtag & Content Strategy
- Task 5: Campaign Effectiveness
- Task 6: Follower Analysis
- Task 7: Video Explanation

## Workflow
1. Data Cleaning and preprocessing of raw datasets
2. Created calculated metrics like engagement rate
3. Used Pivot Tables for analysis
4. Built dashboards for visualization
5. Derived insights and recommendations based on trends

## Key Insights
- Twitter achieved the highest engagemnt rate (~9.6 %) making it the most effective platform for user interaction.
- #SpotifyWrapped is the top-performing hashtag based average clicks, indicating strong audience interest.
- Instagram shows the most consistent weekly follwer growth rate,making it a reliable platform for audeince expansion.
- Twitter and Instagram emerge as the most impactful platforms, while YouTube plays a supporting role in brand visibility.
- Certain content type-platform combinations delivered significantly higher engagement.
- Identified the campaign with the highest ROI (0.0465), highlighting cost-effective marketing strategies.
- Moving average analysis indicates relatively stable follower growth over time, suggesting strong audience retention.
- No strong correlation found between ad spend and engagement, implying that engagement is influenced by multiple external factors.
- 

## Recommendations
- Prioritize Twitter for engagement-driven campaigns and Instagram for sustained audience growth.
- Leverage high-performing hashtags like #SpotifyWrapped to maximize reach and user interaction.
- Focus on platform-specific content strategies:
    - Story-based content for Instagram and Twitter
    - Text-based content for Twitter
    - Video/Reel content for YouTube
- Optimize marketing spend by identifying and investing in campaigns with higher ROI rather than increasing ad spend blindly.
- Improve content quality and relevance, as engagement is influenced more by content strategy than ad spend.
- Maintain consistent posting strategies to sustain steady follower growth and retention.
